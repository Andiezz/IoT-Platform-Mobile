## Description
- Mobile App using [Expo](https://expo.dev)

## Installation

```bash
$ yarn install
$ npm install
```

## Add environment variables
Fill all the environment variables in the **.env.sample** and copy to **.env.local** file for development mode, **.env** for production mode

## Running the app

```bash
$ npx expo start

```
